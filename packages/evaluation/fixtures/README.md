# Evaluation fixtures

Only synthetic, non-authoritative contract fixtures belong here. Authoritative
corpora and run bundles are private, encrypted, content-addressed evaluation
objects with separate human-approved rights records. Never add learner uploads,
assessment data, PII, or production evidence to this directory.

Human listening annotations use `audio-listening-review-v1`; reviewer identities
must be opaque and must not contain names, email addresses, or other PII.
